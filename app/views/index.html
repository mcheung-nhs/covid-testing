Accessibility testing

The GOV.UK prototype kit has been used to create the following pages using the styles, components and patterns found in the GOV.UK Design System.

These have been tested from an accessibility point of view as much as has been possible.

Using the HTML from these pages should fix most of the accessibility issues that exist.
