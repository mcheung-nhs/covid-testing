  3. How do you want to take the coronavirus test?

    Tested link: test-type
    Page with sample HTML: test-type

    Comments

    1. Hint text is not correctly associated with the <fieldset>. Correcting this improves the screen reader experience.
    2. Missing <legend> element which is a requirement inside the <fieldset> element. This can be added and made a heading.
    3. No <main> element on the page. This landmark would help screen reader users with navigation.
    4. If you go back to this page after passing the "What's your name?" page, there are sometimes hidden reCAPTCHA form elements present on this page, which may confuse screen reader users. The elements are listed in the form controls menu when using VoiceOver on iOS and MacOS.

  5. What's your name?

    Tested link: name
    Page with sample HTML: name

    Comments

    1. Missing <legend> element which is a requirement inside the <fieldset> element. This can be added and made a heading.
    2. Incorrect use of the <fieldset> element. Don't need one for each input field. This reduces 'noise' for screen readers
    3. There is no <label> association with the <input> fields
    4. If you go back to this page, there are sometimes hidden reCAPTCHA form elements present on this page, which may confuse screen reader users. The elements are listed in the form controls menu when using VoiceOver on iOS and MacOS.

    References

    • https://design-system.service.gov.uk/patterns/names/
    • https://design-system.service.gov.uk/components/text-input/
